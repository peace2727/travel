| DAY 3: MAY 29 (FRI) | TOUR & DEPARTURE |  |  |  |  |  |  |  |  |  |
| Check out from your hotel and meet with your guide. Begin your day with a visit to Seonjeongneung, the serene royal burial grounds of the Joseon Dynasty, where you can stroll through peaceful pathways and explore the historic tombs surrounded by lush greenery. 
Afterward, head to Imjingak Park and board the DMZ shuttle bus for an insightful tour of the Demilitarized Zone. During the tour, visit Dorasan Observatory, where you can look across the border into North Korea, and explore the historic Third Tunnel of Aggression, one of the infiltration tunnels discovered beneath the border.
After the tour, transfer to Incheon Airport for your departure. |  |  |  |  |  |  |  |  |  |

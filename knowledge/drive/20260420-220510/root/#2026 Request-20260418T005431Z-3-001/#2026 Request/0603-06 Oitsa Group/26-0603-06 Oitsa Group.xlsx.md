| DAY 2: June 4 (THU) - SEOUL |  |  |  |  |  |  |  |  |  |
| Enjoy a full day tour of traditional Seoul. Head to Gwanghwamun Square and walk towards Gyeongbokgung Palace, the first of the five official palaces of the Joseon Dynasty. At the palace, watch the Guard Changing Ceremony. Also visit the National Folk Museum, located on the same grounds of the palace to learn about the traditional Korean life. Continue on to Insadong Street, full of antique shops and art galleries. Next, proceed to Bukchon Hanok Village, a residential area of about 900 traditional hanok houses. Afterwards, stroll through Gwangjang Market, one of Seoul's famous street food markets. Return to the hotel and rest at leisure. |  |  |  |  |  |  |  |  |  |

| Enjoy a full day tour in the modern areas of Seoul. First, visit Bongeunsa Temple, where a 23m Buddha statue stands in Korea's most expensive district. Continue on to Starfield Library, which is a stunning, modern public library located in Seoul's largest underground shopping mall. Then, proceed to Seoul Sky Tower, Korea's tallest building and go up the observatory deck. In the afternoon, visit the National Museum of Korea to learn about South Korean history.

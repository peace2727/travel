| DAY 2: SEOUL |  |  |  |  |  |  |  |  |  |
| Enjoy a full day tour of Seoul. Start from Gwanghwamun Square and walk towards Gyeongbokgung Palace, the first and largest of all five palaces. At the palace, watch the guard changing ceremony and visit the National Folk Museum to learn about the traditions and cultures of the locals. Continue on to Bukchon Hanok Village, home to over 900 traditional Korean houses. Then, visit Insadong Sreet known for its traditional Korean art, antiques, tea houses, and a unique blend of the old and modern charm. Before returning to hotel, stop by Gwangjang Market to explore the traditional market popular for its street food. Then, return to the hotel and rest at leisure. |  |  |  |  |  |  |  |  |  |

| DAY 4: SEOUL - SUWON - SEOUL |  |  |  |  |  |  |  |  |  |
| Visit the outskirts of Seoul today. First stop is the Hwaseong Haenggung Palace, a temporary palace used by the kings when making a visit to the city. Then, head to the Suwon Hwaseong Fortress, a UNESCO World Heritage site featuring an impressive 18th-century fortification with well-preserved defensive walls and structures. Next, proceed to the Korean Folk Village to see what life was like during the Joseon Dynasty and watch a Korean traditional performance. | DAY 5: SEOUL - JEJU |  |  |  |  |  |  |  |  |  |
| Check-out from the hotel and be met by your non-English speaking driver. Take the domestic flight to Jeju Island. Upon arrival, proceed to Haenyeo Museum to learn about the UNESCO Intangible Cultural Heritage designated female divers. Then, visit Seongsan Sunrise Peak, a volcanic crater known for its unique geological formations. Next, head to Jeju Stone Park, a cultural and natural park on Jeju Island that showcases the island's unique volcanic stones, traditional stone sculptures, and the history of Jeju's geology and folklore. | DAY 6: JEJU |  |  |  |  |  |  |  |  |  |
| Visit Hallim Park, a beautifully landscaped garden featuring subtropical plants and lava caves, perfect for a leisurely stroll. Then head to Gotjawal Provincial Park, a dense forest growing on ancient lava flows, known for its rich biodiversity and cool, shaded trails. At the provincial park, try a relaxing foot bath. Continue on to Cheonjeyeon Waterfall, where you can admire a series of three stunning waterfalls and the picturesque Seonimgyo Bridge. Next, visit Jusangjeolli Cliff, where you’ll be awed by the dramatic hexagonal rock formations created by lava meeting the sea. Return to the hotel and rest at leisure. |  |  |  |  |  |  |  |  |  |

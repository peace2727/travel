| DAY 1: April. 27 (Mon) | Start in Seoul → Suwon → Jeonju |  |  |  |  |  |  |  |  |  |
| Starting the day at 9:00 AM, we leave the bustling streets of Seoul behind and head toward Suwon (41km, 1hr 10mins). Our first stop is Hwaseong Fortress, a majestic 18th-century UNESCO World Heritage site where you can wander along the fortress walls and imagine the historic life of the city. Just a short walk away lies Suwon Haenggung Palace, once a temporary residence for kings during their travels, where the elegance of traditional Korean architecture and royal history comes alive.

After a morning immersed in Suwon’s rich history, we continue to Jeonju Hanok Village, arriving in the afternoon (188km, 2hr 30min). Here, the charm of hundreds of traditional Korean houses welcomes you, alongside quaint streets filled with local crafts and the irresistible aroma of Jeonju bibimbap, a culinary highlight of the region. As the day winds down around 6:00 PM, you can enjoy a leisurely stroll, capturing the village’s timeless beauty before the day concludes. |  |  |  |  |  |  |  |  |  |

| DAY 2: April. 28 (Tue) | Tapsa → Gwangju |  |  |  |  |  |  |  |  |  |
| Our day begins at 9:00 AM with a journey to the serene Tapsa Temple at Maisan Mountain  (42km, 60mins). Famous for its unique stone pagodas built without mortar, Tapsa offers a tranquil atmosphere where the harmony between nature and human craftsmanship can be truly appreciated. Walking through the temple grounds, you can feel the peaceful energy and admire the remarkable artistry of each stacked stone pagoda. 

After a morning immersed in Tapsa’s spiritual beauty, we continue our journey to Gwangju, arriving around midday (247km, 3hr 30mins). Here, we explore the city’s historic and cultural highlights starting by passing by the May 18th Memorial Park, honoring the pro-democracy movement, followed by the April Revolution History Hall, which tells the story of Korea’s democratic struggles. If time permits, the tour can continue in Yangnim-dong of historic cultural sites and traditional architecture, offering a glimpse into Gwangju’s rich heritage. The day concludes, and settle in for an overnight stay in Gwangju, ready to continue the journey the next day. |  |  |  |  |  |  |  |  |  |

| DAY 3: April. 29 (Wed) | Mokpo |  |  |  |  |  |  |  |  |  |
| Begin the day with a scenic drive from Gwangju to Mokpo, a historic port city on Korea’s southwest coast. (71km, 60mins) The journey starts at Yudal Mountain, where panoramic views of the harbor, islands, and coastline welcome you to the city. Continue to Mokpo’s Modern History & Culture Street, lined with colonial-era buildings, nostalgic cafés, and colorful murals that tell the story of the city’s past.
Next, experience the lively atmosphere of Dongbu Market, where stalls overflow with fresh seafood, Jeolla-style street snacks, and local delicacies—an ideal spot to enjoy lunch at your own pace. Afterward, discover Gatbawi Rock, a natural formation shaped like a traditional Korean bamboo hat and a beloved symbol of Mokpo’s maritime heritage.
The tour concludes with the Mokpo Marine Cable Car, one of Korea’s longest sea-crossing cable cars. As you glide above the water, take in sweeping views of Yudal Mountain, the bustling harbor, and the scattered islands of the archipelago—an unforgettable highlight of your visit. Then check-in the hotel in Mokpo and free at leisure. |  |  |  |  |  |  |  |  |  |

| DAY 4: April. 30 (Thur) | Suncheon → Boseong → Yeosu |  |  |  |  |  |  |  |  |  |
| After checking out from your hotel in Mokpo, travel east to Boseong, renowned for its endless green tea fields. (90km, 70mins). The Daehan Dawon Plantation stretches across rolling hills, creating a picture-perfect landscape. A gentle walk among the neatly lined tea bushes and a chance to taste Korea’s finest green tea make this an unforgettable stop. 
From Boseong, journey to Suncheon, one of Korea’s ecological treasures.  Explore the Suncheon Bay Wetland Reserve, home to vast reed fields, migratory birds, and winding boardwalks that offer serene views of the tidal flats. Continue to Suncheon Bay National Garden, a beautifully designed landscape where themed gardens from around the world blend with Korea’s natural scenery.
Conclude the day in Yeosu, a coastal city celebrated for its stunning sea views and romantic atmosphere. Stoll the Yi Sunsin Square located in Jungang-dong, Yeosu-si and contains the history and achievements of General Yi Sunsin. The main attraction in the plaza is the life-size turtle ship which you can not only view from the outside but also the inside. Check-in your hotel in Yeosu and stay overnight. |  |  |  |  |  |  |  |  |  |

| DAY 5: May 1 (Fri) | Yeosu |  |  |  |  |  |  |  |  |  |
| Start your day exploring Yeosu, a charming coastal city known for its scenic views and maritime culture. Begin with a visit to Odongdo Island, famous for its camellias, lush walking paths, and panoramic vistas of the surrounding sea. The island’s peaceful trails offer a perfect introduction to Yeosu’s natural beauty.
Next, head to Hyangilam Hermitage, perched on a cliff overlooking the ocean. This historic Buddhist temple provides stunning views of the sunrise and the sparkling coastline, offering a serene and spiritual experience amid dramatic scenery.
Continue to the Yeosu Fish Market, a lively hub where local seafood is sold fresh from the docks. Stroll through the bustling stalls, observe the vibrant fishing culture, and sample some of the city’s renowned delicacies for lunch or light snacks.
After a full day of sightseeing, return to your hotel in Yeosu and relax. |  |  |  |  |  |  |  |  |  |

| DAY 6: May 2 (Sat) | Tongyeong |  |  |  |  |  |  |  |  |  |
| Departing from Yeosu at 9:00 AM, travel along Korea’s scenic southern coastline to reach Tongyeong, a picturesque port city celebrated for its maritime heritage and island views. (123km, 2hrs)
Upon arrival, begin your exploration at Dongpirang Mural Village, a hillside neighborhood adorned with vibrant murals and street art. Wander along the narrow lanes and enjoy colorful scenes depicting local life and culture.
Next, visit Tongyeong Cable Car, ascending to Mireuksan Mountain. From the observation deck, marvel at panoramic vistas of the city, the harbor, and the surrounding archipelago—an iconic spot for capturing the beauty of southern Korea.
Continue to the Tongyeong Jungang Market, a lively traditional marketplace where fresh seafood, local snacks, and artisanal goods abound. Experience the city’s culinary culture and perhaps sample a few local specialties for lunch.
Conclude the day with a relaxing check-in at your hotel in Tongyeong, enjoying the evening ambiance of this charming coastal city. An overnight stay allows you to unwind and prepare for the next day’s adventures along Korea’s scenic southern coast. |  |  |  |  |  |  |  |  |  |

| DAY 7: May 3 (Sun) | Oedo Botanica → Busan (end of tour) |  |  |  |  |  |  |  |  |  |
| Start a day with visiting the Hill of Wind" where serves as a captivating first destination in Geoje City, renowned for its tranquil landscapes and natural beauty. With charming scenery and a variety of attractions, it makes for an appealing starting point for visitors exploring the area. Don't miss the opportunity to check out the local tourist information if it's your first stop. 
The Dojangpo Ferry is a popular boat service in the Gyeongsangnam-do region of South Korea. Departing from Dojangpo, it offers a scenic journey along the beautiful coastline and traditional fishing villages. You can enjoy the picturesque blend of blue sea and distinctive rocky hills, experiencing the natural beauty and culture of the region. The Dojangpo Ferry provides a unique opportunity for tourists and travelers to explore the charm of this area.
Haegeumgang Lion Rock" is likely a rock formation near Haegeumgang in South Korea. Haegeumgang is known for its beautiful coastal landscapes, and the Lion Rock is one of its distinctive rock formations.
Oedo Botania" is often considered a captivating final destination for visitors exploring the southern coast of South Korea, particularly near Geoje Island. This enchanting island is celebrated for its stunning botanical garden, which showcases a diverse array of plants and flowers amidst scenic landscapes and coastal cliffs. The journey to Oedo Botania often involves a ferry ride from Geoje Island, offering travelers not only a destination but also a picturesque and memorable coastal experience. Finally head to Busan, the second largest ciy in Korea. The tour will end with drop-off at your preferred location, whether it is your hotel, Busan Station, or Gimhae Airport. |  |  |  |  |  |  |  |  |  |

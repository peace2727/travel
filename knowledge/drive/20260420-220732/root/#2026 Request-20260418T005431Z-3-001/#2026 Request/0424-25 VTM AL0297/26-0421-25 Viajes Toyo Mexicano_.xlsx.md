| DAY 1: April 21 (TUE) | ARRIVAL |  |  |  |  |  |  |  |  |  |
| Welcome to South Korea! Upon arrival, meet with your guide at Incheon Airport. Transfer to Seoul and explore Gwangjang Market, one of Seoul's famous street food markets. Next, stop by Heunginjimun, Seoul’s historic East Gate, a striking remnant of the Joseon Dynasty city walls that beautifully blends ancient architecture with the surrounding modern cityscape. Continue with a relaxing stroll along Cheonggyecheon Stream, a peaceful urban oasis flowing through downtown Seoul, where walking paths, bridges, and gentle water features offer a refreshing contrast to the bustling city above. Then, transfer to the hotel for check-in and rest at leisure. |  |  |  |  |  |  |  |  |  |

| DAY 2: April 22 (WED) | SEOUL |  |  |  |  |  |  |  |  |  |
| Start the day at Gwanghwamun Square and walk towards Gyeongbokgung Palace, the first and largest of all five palaces. At the palace, watch the guard changing ceremony and visit the National Folk Museum to learn about the traditions and cultures of the locals. Next, stroll through the charming alleyways of Bukchon Hanok Village, where hundreds of well-preserved traditional Korean houses (hanok) sit on gentle hills between modern neighborhoods, offering a glimpse into Seoul’s past. Continue on to Insadong, a cultural district known for its antique shops, art galleries, traditional tea houses, and souvenir stores showcasing Korean craftsmanship. Just nearby is Ikseondong, a trendy yet nostalgic area where renovated hanok buildings house stylish cafés, boutique shops, and unique eateries, blending old-world charm with modern creativity. Then, return to the hotel and rest at leisure. |  |  |  |  |  |  |  |  |  |

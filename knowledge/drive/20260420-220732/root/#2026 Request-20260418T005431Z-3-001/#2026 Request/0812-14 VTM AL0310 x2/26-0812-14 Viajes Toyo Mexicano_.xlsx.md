| DAY 1: Aug 12 (WED) | SEOUL CITY TOUR |  |  |  |  |  |  |  |  |  |
| Start the day at Gyeongbokgung Palace, the first and largest of all five palaces. If possible (depends on the weather), watch the guard changing ceremony at the palace and visit the National Folk Museum to learn about the traditions and cultures of the locals. Next, stroll through the charming alleyways of Bukchon Hanok Village, where hundreds of well-preserved traditional Korean houses (hanok) sit on gentle hills between modern neighborhoods, offering a glimpse into Seoul’s past. Continue to Insadong Street, known for antiques, traditional crafts, and unique souvenirs. Then, experience the bustling Gwangjang Market, where you can enjoy a variety of traditional Korean street foods. Return to your hotel and rest at leisure. |  |  |  |  |  |  |  |  |  |

| SEOUL - JUNE 12 (FRI) |  |  |  |  |  |  |  |  |  |
| Meet your Spanish-speaking guide in front of the hotel. 
Begin the day at Gwanghwamun Square, walking towards Gyeongbokgung Palace, the first and largest of Seoul's five royal palaces. At the palace, watch the guard changing ceremony and visit the National Folk Museum to learn about the traditional culture and daily life of the locals. Continue on to Insadong Sreet, famous for traditional art, antiques, tea houses, and its unique blend of the old and modern charm. 

For a full-day tour, proceed to Bukchon Hanok Village, where elegant hanok homes line quiet streets between Seoul’s palaces and vibrant city life. Then, explore Gwangjang Market and discover its popular street food. Return to the hotel and rest at leisure. |  |  |  |  |  |  |  |  |  |

| BUSAN - JUNE 15 (MON) |  |  |  |  |  |  |  |  |  |
| Meet your Spanish-speaking guide in front of the hotel. 
Begin the day at Haedong Yonggungsa Temple, a rare seaside sanctuary offering breathtaking ocean views and a peaceful atmosphere. Continue with a walk on the Cheongsapo Sky Deck for scenic coastal vistas. 

For a full-day tour, visit Jagalchi Fish Market and Gukje Market, two of Korea’s largest traditional markets, before discovering Gamcheon Cultural Village, a colorful hillside community famed for its maze-like alleys, vibrant murals, and charming art shops that reflect Busan’s creative spirit and history. Return to the hotel and rest at leisure. |  |  |  |  |  |  |  |  |  |

| GYEONGJU - JUNE 17 (WED) |  |  |  |  |  |  |  |  |  |
| Meet your Spanish-speaking guide in front of the hotel. 
Visit Bulguksa Temple and Seokguram Grotto, UNESCO World Heritage sites renowned for their exquisite Buddhist architecture, stone craftsmanship, and serene mountain setting. 

For a full-day tour, continue to Gyeongju National Museum to explore priceless Silla artifacts, stroll through the Royal Tumuli Park with its ancient royal tombs, and admire Cheomseongdae Observatory, the oldest surviving astronomical observatory in East Asia.
